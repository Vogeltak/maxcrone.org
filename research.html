Multi-core flow classification methods for load balancing

[PDF] 63-page report

This document is the product of a semester-long group project during my time at KTH, Stockholm. We implemented flow tables that scale to multiple cores, and thoroughly evaluated their performance. The report provides the results as a detailed comparison.

I worked with the FastClick framework for composing network functions, and used the Data Plane Development Kit (DPDK) underneath for packet processing in user space. Traffic generation was built using TRex.
